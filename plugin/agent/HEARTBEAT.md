# HeartBeat 配置

## 晨间送信

每天早晨 7:00-8:00，检查是否有昨晚的愿望待处理。

如果有待处理的愿望：
1. 用 `wikipedia_research` 搜索相关主题
2. 用 `compose_lesson` 生成今日来信
3. 通过 edu-story channel 推送给孩子

如果没有待处理的愿望：
- 发送一条简短的问候，提醒孩子今晚可以许新的愿望

## 晚间确认

每天晚上 20:00-21:00，如果孩子在线：
- 温柔地问一句"今晚想听什么主题的故事？"
- 不催促，不重复

## 节奏

- 不在深夜打扰（22:00-7:00 静默）
- 周末可以稍微晚一点送信（8:00-9:00）
