# HeartBeat 配置

## 晚间送信（默认 20:30，可配置）

每天在孩子配置的送信时间检查是否有待处理的愿望。

如果有待处理的愿望：
1. 用 `get_weather` 获取今日天气
2. 用 `get_learner_session` 查看孩子的学习状态和词库
3. 用 `wikipedia_research` 搜索相关主题，优先使用简洁英文 research query，最多尝试 2 次
4. 用 `compose_lesson` 生成今日来信（信件 + 场景 + 词卡）
5. 信件自动通过 capybara-letter channel 推送给孩子

如果没有待处理的愿望：
- 根据孩子的兴趣和最近学过的内容，自主选择一个主题
- 同样走完整的 research → compose 流程
- 发送一条简短的问候，告诉孩子今天的信已经到了

## 晚间互动窗口（送信前后 1 小时）

如果孩子在线并且还没许愿：
- 温柔地问一句"今晚想让我明天带什么主题的信回来？"
- 不催促，不重复

## 节奏

- 不在深夜打扰（22:00-7:00 静默）
- 默认送信时间是 20:30，但以家长配置为准
- 如果孩子连续 3 天没上线，下次见面时先关心再送信
